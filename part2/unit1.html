Unit 1 (HTML & CSS)

HTML5

HTML stands for Hyper Text Markup Language. It is the standard markup language for creating Web pages HTML describes the structure of a Web page and consists of a series of elements. HTML elements tell the browser how to display the content also its elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

nav:

Simply defines navigation links and helps us navigate between different pages. <nav> Navigation Links </nav>

header:

funstions as a header for a document or section of the page. <header> Document Header </header>

footer:

Basically acts as a footer for a document or section of the page. <footer> Document Footer </footer>

section:

This is where the general information of the page is usually stored. <section> Document Footer </section>

article:

Article is where the more detailed information about the page is stored. <article> Document Footer </article>

Input Type Text:

Takes the text from the user through the input form



html text: <form> <label>Name:</label><br> <input type="text"><br> </form>

Input Type Password:

Takes the password from the user through the input form and hides it



html text: <form> <label>Password:</label><br> <input type="password"><br> </form>

Input Type Submit:

A submit button which would be configured to trigger a submission of the form


html text: <form> <input type="submit" value="Submit"><br> </form>

Input Type Radio:

You can only choose one of the radio buttons





html text: <form> <label>Choose your favorite unit:</label><br> <input type="radio" value="unit1"> <label>Unit 1 (HTML & CSS)</label><br> <input type="radio" value="unit2"> <label>Unit 2 (JavaScript)</label><br> <input type="radio" value="unit2"> <label>Unit 2 (XML and Ajax)</label><br> </form>

Input Type Checkbox:

You can choose zero or more of the boxes





html text: <form> <label>Choose your favorite unit:</label><br> <input type="checkbox" value="unit1"> <label>Unit 1 (HTML & CSS)</label><br> <input type="checkbox" value="unit2"> <label>Unit 2 (JavaScript)</label><br> <input type="checkbox" value="unit2"> <label>Unit 2 (XML and Ajax)</label><br> </form>

Input Type Color:

You can select a color

CSS

CSS stands for Cascading Style Sheets and describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files

Text Color:

"color" attribute specifies the color of text

my color is blue!

css text: color: blue;

Background Color:

"background-color" attribute specifies the background color of text

my background color is grey!

css text: background-color: grey;

Border Color:

"border-color" attribute specifies the border color of text, note that we must also include the border size

my border color is red!

css text: border: 2px solid red;

Text Font Family:

"font-family" attribute specifies the family of the font

Times new roman font!

css text: font-family: "Times New Roman", Times, serif;

Text Font Size:

"font-size" attribute specifies the size of the font

Times new roman font size 24px!

css text: font-family: "Times New Roman", Times, serif; font-size: 24px;

Text Alignment:

"text-align" attribute specifies the alignment of the text, default text is aligned to the left

aligned to the right

aligned to the center

css text: text-align: right;

css text: text-align: center;

Selectors:

If you take a look at the previous example it might confuse you as why css text for both cases of alignment is actually the same! We use "selectors" to distinguish between these elements!

For example, to apply an attribute to all the paragraphs we do the following: p {color: blue} Now all the paragraphs will be shown in blue text if we apply this

To distinguish between different elements of the same type we can use classes or ids

To assign a class "blue" to a certain paragraph we do the following: <p class="blue">Paragraph</p>

Now in css only add a "." before the class name to apply a certain style only specific to any element with class "blue"

css text: .blue {color blue}

Paragraph

It is almost the exact same way with ids, only the way to declare them is a bit different. Let's assign a "red" id to a paragraph: <p id="red">Paragraph</p>

In css we just add "#" before the id name, this will only apply to elements with id = "red"
