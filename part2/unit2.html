Unit 2 (JavaScript)

JavaScript

often abbreviated as JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. As of 2022, 98% of websites use JavaScript on the client side for webpage behavior, often incorporating third-party libraries. All major web browsers have a dedicated JavaScript engine to execute the code on users' devices.

Variables

Different Variable Declarations:

  • var: variable
  • let: variable which the type cannot change
  • cont: contant type

You can declare a variable by simply writing the following: var a = 1; By doing this we are assinging the value 1 to variable a.

Any type of data can be assigned using this method including strings. For example, var s = "Hello World!";

Operators

Different Operators:

  • +: addition
  • -: subtraction
  • *: multiplication
  • /: division

Example var a = 1; var b = -1; var c = a + b;

In this simple arithmetic operation we get that c = 0!

Arrays

We can have a list of variables by using arrays. For example, we can have a list of colors:

const colors = ["red", "green", "blue", "purple"]

If Else Statements

Imagine a simple case which you want your program to work differently based on different conditions, we do this using if else statements

For example, if a > 0 I want to assing the value 1 to b otherwise 0

We write the following code: if(a > 0) { b = 1 } else { b = 0 }

For Loops

When we want to iterate over a sequence and perform a certain action, we use for loops. For example, if I want to write "Hello World!" 5 times to the console, we do the following:

for(var i = 0; i < 5; i++){ console.log("Hello World!") }

Functions

Functions basically can take an input and perform an action. For example, if I want to add 1 to the input:

function add_one(n){ return n + 1; } add_one(1), this statement would return the value 2 to us!

Objects

We can define an object and give it many properties. For example, a car that has a certain color, speed, price.

const car = {speed:"150", price:"50,000$", color:"white"};

We can access these properties by simply writing this: car.price!
